EmojiPicker
.js v2

A flexible, event-driven emoji picker. Zero dependencies. Full developer control. Now with IndexedDB caching, favorites, i18n, emoji support detection and TypeScript.

Live Demos

All examples share the same v2 library. Data loads once, caches in IndexedDB forever.

Example 01
Insert into text input at cursor
const picker = new EmojiPicker({ container: "#btn-chat", dataUrl: "/emoji-data.json" }) picker.on("emojiClick", (emoji) => { input.value += emoji.char })
